## 💻 Projeto

O Biscoito da Sorte foi desenvolvido para estudar e aplicar tudo aprendido no curso até então. Entre os temas mais presentes estão:

- Estrutura de dados HTML
- Animações com CSS
- Funções no Javascript
- Manipulação da DOM
- Biblioteca JS Math()
- Funções *callback*
- Arrays
